# Sistema de Controle de Veículos

## Descrição
O **Sistema de Controle de Veículos** foi desenvolvido para facilitar o gerenciamento de viagens e a organização de frotas. Ele oferece funcionalidades simples e eficazes, como cadastro, acompanhamento e finalização de viagens, tudo em uma interface moderna e responsiva.

Este projeto utiliza tecnologias populares e acessíveis, como **PHP**, **MySQL** e **Bootstrap**, garantindo facilidade na implementação, manutenção e expansão.

---

## Funcionalidades
- **Cadastro de Viagens**: Adicione informações detalhadas sobre veículos, motoristas, destinos e horários de saída.
- **Visualização de Registros**: Consulte viagens em andamento ou concluídas de forma organizada em uma tabela responsiva.
- **Finalização de Viagens**: Registre horários de chegada para completar os dados da viagem.
- **Design Responsivo**: Interface adaptável a diferentes dispositivos (computadores, tablets e smartphones).

---

## Tecnologias Utilizadas
- **PHP**: Backend responsável pela lógica de negócios.
- **MySQL**: Banco de dados para armazenamento seguro e eficiente das informações.
- **Bootstrap 5**: Para design moderno e responsivo.
- **HTML e CSS**: Para estruturação e personalização da interface.

---

## Requisitos do Sistema
- Servidor local (como XAMPP ou WAMP) com suporte para PHP e MySQL.
- Navegador moderno para acessar a interface web.
- Banco de dados configurado com as tabelas necessárias.

---

## Como Utilizar
1. Clone o repositório ou baixe o código-fonte.
2. Configure o banco de dados:
   - Crie um banco de dados MySQL com o nome `controle_veiculos`.
   - Importe o arquivo `controle_veiculos.sql` (se fornecido) para criar as tabelas.
3. Ajuste as credenciais de banco de dados no código PHP, caso necessário.
4. Execute o projeto em um servidor local.
5. Acesse a aplicação pelo navegador (exemplo: `http://localhost/controle_veiculos`).
